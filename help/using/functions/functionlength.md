# length {#length}

Retourneert het aantal tekens van de tekenreeksexpressie in de parameter.

## Categorie

Tekenreeks

## Functiesyntaxis

`length(<parameters>)`

## Parameter

* string

## Handtekening en type geretourneerd

`length(<string>)`

Retourneert een geheel getal.

## Voorbeeld

`length("Hello World")`

Retourneert 11.
