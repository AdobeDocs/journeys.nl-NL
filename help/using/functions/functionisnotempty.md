# isNotEmpty {#isNotEmpty}

Retourneert true als de tekenreeks in de parameter niet leeg is.

## Categorie

String

## Functiesyntaxis

`isNotEmpty(<parameters>)`

## Parameters

* string

## Handtekening en type geretourneerd

`isNotEmpty(<string>)`

Retourneert een Booleaanse waarde.

## Voorbeeld

`isNotEmpty("")`

Retourneert false.

`isNotEmpty("hello")`

Retourneert true.
