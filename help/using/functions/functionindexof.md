# indexOf {#indexOf}

Controleert of de eerste argumenttekenreeks eindigt met een specifieke tekenreeks (tweede argumenttekenreeks). Retourneer -1 als de tekenreeks niet wordt gevonden.

## Categorie

Tekenreeks

## Function syntax

`indexOf(<parameters>)`

## Parameters

| Parameter | Type |
|-----------|------------------|
| string | Tekenreeks |
| specified value | Tekenreeks |

## Handtekening en type geretourneerd

`indexOf(<string>,<string>)`

Retourneert een geheel getal.

## Voorbeeld

`indexOf("Hello", "l")`

Retourneert 2.

Explanation:

In &quot;Hello&quot;, the first occurrence of “l” is at position 2.
