# round {#round}

Retourneert de dichtstbijzijnde gehele-getalwaarde voor het argument met bindingen die worden afgerond naar positief oneindig.

## Categorie

Math

## Functiesyntaxis

`round(<parameters>)`

## Parameters

* decimaal
* integer

## Handtekeningen en type geretourneerd

`round(<decimal>)`

`round(<integer>)`

Retourneer een geheel getal.

## Voorbeeld

`round(3.14)`

Retourneert 3.

`round(3.54)`

Retourneert 4.

`round(-3.14)`

Retourneert -3.

`round(3)`

Retourneert 3.
