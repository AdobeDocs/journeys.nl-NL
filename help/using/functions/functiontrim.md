# bijsnijden {#trim}

Hiermee verwijdert u de begin- en eindspaties.

## Categorie

Tekenreeks

## Functiesyntaxis

`trim(<parameters>)`

## Parameter

| Parameter | Type |
|-----------|------------------|
| string | string |

## Handtekening en type geretourneerd

`trim(<string>)`

Retourneer een tekenreeks.

## Voorbeeld

`trim(" Hello ")`

Retourneert &quot;Hello&quot;.
