# upper {#upper}

Retourneert een hoofdletterversie van de parameter.

## Categorie

Tekenreeks

## Functiesyntaxis

`upper(<parameters>)`

## Handtekening en type geretourneerd

`upper(<string>)`

Retourneer een tekenreeks.

## Voorbeeld

`upper("b")`

Retourneert &quot;B&quot;.
